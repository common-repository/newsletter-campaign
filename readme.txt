== Installation ==

1. Upload `newsletter-campaign` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in your WordPress admin panel.
3. You can adjust necessary settings through your WordPress admin panel in "Mailchimp Campaign" > "Configuration".
